


		В JS есть такой ньюанс при работе с массивами и обьектами. 

		Пример: 

		let a = {name: "123"}

		let b = a

		b.name  = "Edk"

		console.log(a.name);	// Мы получим "Edk" вместо "123", но мы не трогали первый обьект.




		Это происходит потому-что - присваивание(приравнивание) массивов и обьектов происходит по сслыке. В нашем случае обьекты a и b ссылаются на один и тот же обьект. Поэтому, когда мы меням один, также меняется второй...

		Чтобы избежать этого нам придется скопировать обьект следующими методами:

		Пример: 	

			let a = {name: "Eddie"}

			1) let b = Object.assign({}, a);		// Пустой обьект мы должны прописать первым, таким образом мы меняем ссылку на пустой обьект.

			2) let b = {... a}				// Копирование с помощью оператора spread

			3) let b = JSON.parse(JSON.stringify(a))	// В этом случае мы превращаем обьект в json потом в новый обьект.

			PS - в случае если обьект содержит вложенные обьекты и массивы правильно сработает только 3-ий способ.
