
 								Рекурсия 


	Рекурсия это вызов функции внутри себя		Пример:  function rec() { a++   rec()  }; // В данном случае код будет выполняться бесконечно.

	С помощью рекурсии можно некторый код намного облегчить.   

	PS - Важно предвидеть выход из данного замкнутого круга.

	Пример:


	
	Запись с помощью цикла:

		var a = 1	
	
		function improveVar(var, value) {

						for (var i = 0; i < value; i++) {

										var++

										if (var == value) {

													return

												   }

										}



	
	Запсиь с помощью рекурсии :

		var a = 1

		function improveVar(value) {

					a++ 

					if (a < value) {
				
							improveVar(value)

							}

					    }


	P.S. - в рекурсию также можно добавлять setTimeOut() и вставить в него саму функцию и тогда цикл рекурсии будет происходить с интервалом.
