	
				JavaScript Свойства



	Все значения в JavaScript, за исключением null и undefined, 

	содержат набор вспомогательных функций и значений, доступных через точку либо переменная в скобках.


  	var name = "Eduard"

	name.length     //  Выдает число 6







Свойства которые я знаю:



1) string/array.slice(firstItem, lastItem) - Выдает кусок строки или массива от начального до конечного значения. Если указать 1 значение.

Выдаст значение от 1 до конца строки. 

Данное свойство работает только со строками и массивами.




2) variable.length - Выдает число которое обозначает кол-во знаков переменной(строки),кол-во элементов массива.

Также работает только со строками и массивами.








3) Math.floor(number); Свойство которое, дает округленное числа до (пола) нижнего значения (не как в математике) т.е. 0.75 = 0        3.3 = 3            4.5 = 4        6.99 = 6

4) Math.ceil(number): Свойство которое, дает округленное числа до (потолка) верхнего значения (не как в математике) т.е. 0.8 = 1     0.1 = 1             3.9 = 4       5.01 = 6

5) Math.round(number); Свойство которое, дает округленное числа точно как в математике. 45.5 = 46     //    45.2 = 45

6) Math.pow(x,y); -  Свойство, которое дает число со степенью. x - основание, y - степень.

7) Math.random(); - Метод который выдает рандомное дробное число от 0 до 1. С 16 цифрами после 0. 0.0000000000000001 , 0.9999999999999999.    P.S. - в скобках не указываем ничего











8) element.innerHTML = "" - с помощью .innerHTML мы можем менять значение внутри html элемента(-ов) на который ссылаемся, в кавычках указываем данные которые хотим ввести.

	Пример: document.querySelector(".intro-text").innerHTML // "Welcome to our website !"

	ps - С помощью данного способа мы меняем весь html код в теге (Не только контент)          Пример: heading.innerHTML = "<span>Eduard</span><b>Kosyan</b>"


9) element.textContent - Выдает строку(string) только с текстом(контентом), который содержит данный элемент.





10) element.style.property  - мы можем получить или изменить значение CSS свойства элемента.

			Пример: element.style.backgroundColor = red; 	// Установка фона в красный

				var color = element.style.backgroundColor;  	// Получение значения свойства 

			P.S. - CSS свойства из 2 и более свойст записываются в camelCase.





11) element.classList , выдает лист с классами которые содержит html элемент.





12) element.attributes - Выдает лист с аттрибутами html элемента.





13) element.getAttribute("attribute") - Получение значения аттрибута , которое мы указали в скобках.       Пример: document.querySelector(".item__link").getAttribute("href") - выдает значение: "www.example.ru"







14) Обьект window - window Обьект который содержит много свойств об окне браузера. Мы можем прописать к нему свойства после точки "." и получить разные значение (скролл топ ...)







15) window.scrollY  - Выдаст значение вертикального скролла окна браузера в пикселях.              Также есть   window.scrollX 

	Также можем изменить задав:   window.scrollTop = 900;




16) element.offsetTop  - Выдает растояние от верхнего края элемента до верхнего края документа.






18) input.value  - Выдает значение input-а.             Можем изменить - input.value = 99;







68) Координаты события(клика)                     element.on("click", function(event) { // code})       event.screenX Кординаты X в экране монитора.

													event.clientX Кординаты X в window.

													event.pageX   Кординаты X в html документе.
