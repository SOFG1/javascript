



			Object.create()

	.create() - метод у глобального класса Object. Который создает новый обьект, принимает обьект, которого делает прототипом первого уровня у создаваемого. 

	Также может содержать создаваемый обьект может содержать инкапсулированные приватные свойства, свойства с другими особенностями и getter-ы и setter-ы для приватных свойств.

	Пример: Object.create(protoObject,newObject);

	PS - если ввести один параметр создасться пустой обьект с введенным прототипом.
	
	!PS - Также значения нового обьекта должы быть также обьектом, а значение должно ялвляться значением ключ.слова - value в этом обьекте.

	Пример: const me = Object.create(protoObject, {

							name: {
								value: "Eddie"
							       },
							birthYear: {
								value: 2000
							       },
						      });





		У этого способа есть некоторые отличия:


	
	1) Если мы выведем в консоль обьект person, мы увидим что значения добавленные через Object.create() полупрозрачные. 
	   Это озночает - что они есть, но если мы применим цикл к обьект то эти значения считываться не будут.





	2) Значения нового обьекта декларированные в Object.create() не могут переинициализироваться(перезаписыватсья) (по умолчанию)



	3) Обьект, который записывается в значании (содержащий "value") может содержать параметры, которые переопределяют свойства значений.


		Параметр enumerable - позволяет своему значению быть исчисляемым и работать в цикле.

		Параметр writable - позволяет своему значению перезаписываться.

		Параметр configurable - позволяет удалять это значение в род. обьекте.

	Пример: const person = Object.create({},{
						 name: {
								value: "Eddie",
								enumerable: true,
								writable: true,
								configurable: true,
							}
						 });

	Пример удаления значения с именем: delete person.birthYear;		// Используем оператор delete




	Также можем использовать getter-ы и setter-ы для свойст обьекта.

	Пример: const person = Object.create({},{
						 name: {
								value: "Eddie",
								enumerable: true,
								writable: true,
								configurable: true,
								get() { // getter for person.name },
								set() { // setter for person.name },
							}
						 });
