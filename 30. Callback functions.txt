


			Callback функция

		Callback функция имеет несколько определений.
	

	1) Функция, которая выполняется только после завершения другой функции называется.

	2) Функции, которые передаются как аргумент.

	3) Функция которая передается в другой js файл или обьект. Для выполнения.

	PS - У callback-ов очень часто меняется контекст вызова. Поэтому в этих случаях нужно использовать .bind()
	

	С помощью callback функции можно делать экспорт с помощью импорта(предотвращение цикличной зависимости)


		Прмиер:

		//First file //////////////////
		
		let a = 1;

		export let equal = (data) { a = data    }

		// ./First file ///////////////



		// Second file //////////////////

		import {equal} from './first'
		
		let b = 2;

		equal(b)

		// ./ Second file ///////////////


	В данном примере мы экспортировали b из второго файла. Но теоритически мы импортируем из первого файла.
