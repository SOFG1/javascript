			
				JavaScript функции

	Функции по сути обьекты, которые содержат в себе код который будет выполняться.








	Декларирование функции: 1) function showMessage(inputs) {					// Function Declaration Когда декларируем функцию глобально, можем использовать также до декларации.

								var string = "something";

								// Code 

								return string; // Выполнение функции выдает переменную string

								}




	
				2) var showSomething = function(inputs) {				// Function Declaration Когда декларируем функцию глобально, в переменную, можем использовать только после декларации.

								var ret = "something";

								// Code 

								return ret // Выполнение функции выдает переменную ret

								}			
				


	Выполнение функции   showMessage(inputs); ,    showSomething();         	// Function Expression











1) Функции можно вкладывать друг в друга.  НО функция декларированная в другой функции не будет доступна за пределами родительской функции.


2) Функция вложенная в саму себя называется рекурсией. см. #10. Рекурсия

















3)					Стрелочная функция.

	Существует также стрелочный синтаксис декларирования безымянной функции в переменную:


	let functionVar = (functionParams) => //code ;	===  let functionVar = function(functionParams) { //code }                // Это однострочная стрелочная функция


	let functionVar = (functionParams) => { //code										// Это многострочная стрелочная функция мы должны завернуть код в фиг. скобки.

						//code

						//code      
					  	
						}

	Отличия стрелочной функции:

		1) Не имеет контекста выполнения

		2) Нельзя обращаться к параметрам с помощью ключ. слова 'arguments'

		3) если размещена в пределах одной строки - сразу будет возвращать содержимое без ключ. слова 'return'

		4) если размещена в пределах одной строки - можно опустить фигурные скобки		// (params) => // code







4) arguments - ключевое слово arguments в функции, выдает обьект с параметрами данной функции.   PS - не работает в стрелочной функции.
